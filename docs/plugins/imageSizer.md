# افزونه تنظیم اندازه تصویر (Image Sizer)

این افزونه یک قابلیت بسیار کاربردی را به شیرازه اضافه می‌کند: امکان تنظیم اندازه (عرض و ارتفاع) تصاویر مستقیماً از طریق سینتکس مارک‌داون. این ویژگی به نویسندگان محتوا کنترل بیشتری روی چیدمان و ظاهر صفحاتشان می‌دهد.

## ویژگی‌های اصلی

-   **سینتکس ساده و خوانا:** ابعاد تصویر مستقیماً پس از سینتکس تصویر در آکولاد (`{}`) قرار می‌گیرد.
-   **انعطاف‌پذیری بالا:** پشتیبانی از تعیین عرض و ارتفاع، فقط عرض، فقط ارتفاع، یا اندازه یکسان برای هر دو.
-   **پشتیبانی از اعداد فارسی:** شما می‌توانید ابعاد را با اعداد فارسی (مانند `{۳۰۰x۲۰۰}`) نیز مشخص کنید.
-   **واحد پیش‌فرض پیکسل:** نیازی به نوشتن `px` نیست.
-   **عملکرد هوشمند:** افزونه به صورت خودکار سینتکس را از صفحه حذف می‌کند تا متن اضافی نمایش داده نشود.

## فعال‌سازی افزونه

این افزونه به صورت پیش‌فرض در فایل `config/config.js` فعال است. برای فعال یا غیرفعال کردن آن، کافیست مسیر `src/plugins/imageSizer/imageSizer.js` را در آرایه `plugins` نگه دارید یا حذف کنید.

```javascript
// in config/config.js
window.shirazeh = {
  // ...
  plugins: [
    'src/plugins/imageSizer/imageSizer.js' // فعال‌سازی افزونه
  ],
  // ...
};
```

## پیکربندی

تنها گزینه پیکربندی برای این افزونه، فعال یا غیرفعال کردن آن است که از طریق آبجکت `imageSizer` در فایل `config.js` انجام می‌شود.

```javascript
// in config/config.js
window.shirazeh = {
  // ...
  imageSizer: {
    enabled: true, // برای غیرفعال کردن، false کنید
  },
  // ...
};
```

## نحوه استفاده (سینتکس)

برای استفاده از این افزونه، کافیست ابعاد مورد نظر را داخل یک آکولاد `{}` بلافاصله بعد از سینتکس استاندارد تصویر در مارک‌داون قرار دهید.

### ۱. تعیین عرض و ارتفاع

`![توضیح](path/to/image.jpg){عرضxارتفاع}`

- **مثال:** `![لوگو](logo.png){300x200}`

### ۲. تعیین فقط عرض (ارتفاع خودکار تنظیم می‌شود)

`![توضیح](path/to/image.jpg){عرضx}`

- **مثال:** `![بنر](banner.png){800x}`

### ۳. تعیین فقط ارتفاع (عرض خودکار تنظیم می‌شود)

`![توضیح](path/to/image.jpg){xارتفاع}`

- **مثال:** `![آواتار](avatar.jpg){x150}`

### ۴. تعیین اندازه یکسان برای عرض و ارتفاع

`![توضیح](path/to/image.jpg){اندازه}`

- **مثال:** `![آیکون](icon.png){64}`

### مثال با اعداد فارسی

افزونه به طور کامل از اعداد فارسی پشتیبانی می‌کند.

`![لوگوی شیرازه](/assets/icon/shirazehLogo.png){۱۲۰x}`
