# افزونه لینک گیت‌هاب (GitHub Corner)

این افزونه یک لینک زیبا و شناخته‌شده به شکل "روبان" در گوشه صفحه اضافه می‌کند که مستقیماً به مخزن گیت‌هاب پروژه شما لینک می‌دهد. این یک راه عالی برای تشویق کاربران به مشاهده کد منبع، مشارکت در پروژه یا گزارش مشکلات است.

## ویژگی‌های اصلی

-   **پیکربندی آسان:** تمام تنظیمات از طریق فایل `config.js` قابل مدیریت است.
-   **یکپارچگی با سیستم ویجت:** موقعیت، اولویت و رفتار واکنش‌گرای این کامپوننت به طور کامل از طریق **[سیستم مدیریت ویجت‌ها](/docs/widgetSystem)** کنترل می‌شود.
-   **رنگ‌بندی سفارشی:** رنگ پس‌زمینه و آیکون به صورت خودکار با تم (روشن/تاریک) هماهنگ می‌شود، اما شما می‌توانید آن‌ها را به دلخواه خود بازنویسی کنید.
-   **انیمیشن جذاب:** هنگام هاور کردن ماوس روی آیکون، انیمیشن کلاسیک `octocat-wave` (تکان خوردن دست اختاپوس) اجرا می‌شود.
-   **حالت دکمه‌ای هوشمند:** اگر این ویجت را در یک اسلات استاندارد (مانند نوار ناوبری) قرار دهید، به صورت خودکار به یک دکمه گرد و زیبا تبدیل می‌شود.

## فعال‌سازی افزونه

برای فعال کردن این افزونه، ابتدا مسیر آن را به لیست `plugins` در فایل `config.js` اضافه کنید:

```javascript
// in config/config.js
window.shirazeh = {
  // ...
  plugins: [
    'src/plugins/githubCorner/githubCorner.js'
  ],
  // ...
};
```

سپس، بخش `githubCorner` را در همان فایل پیکربندی و فعال کنید.

## پیکربندی

تنظیمات اصلی این افزونه از طریق آبجکت `githubCorner` در فایل `config.js` انجام می‌شود.

```javascript
// in config/config.js
window.shirazeh = {
  // ...
  githubCorner: {
    enabled: true,
    url: 'https://github.com/barnevis/shirazeh',
    backgroundColor: '',
    iconColor: '',
  },
  // ...
};
```

### گزینه‌های پیکربندی

-   `enabled`:
    -   **نوع:** `boolean`
    -   **مقدار پیش‌فرض:** `false`
    -   **توضیحات:** با `true` کردن این گزینه، افزونه فعال می‌شود.

-   `url`:
    -   **نوع:** `string`
    -   **مقدار پیش‌فرض:** `''`
    -   **توضیحات:** این گزینه **الزامی** است. آدرس کامل مخزن گیت‌هاب خود را در این فیلد وارد کنید. اگر این فیلد خالی باشد، افزونه نمایش داده نخواهد شد.

-   `backgroundColor`:
    -   **نوع:** `string`
    -   **مقدار پیش‌فرض:** `''` (خالی)
    -   **توضیحات:** رنگ پس‌زمینه آیکون را مشخص می‌کند. اگر این فیلد خالی باشد، افزونه به صورت خودکار از رنگ اصلی تم (`--primary-color`) استفاده می‌کند تا با ظاهر ویکی شما هماهنگ باشد. شما می‌توانید هر مقدار رنگ معتبر CSS (مانند `'#151513'` یا `'rgb(21, 21, 19)'`) را در اینجا وارد کنید.

-   `iconColor`:
    -   **نوع:** `string`
    -   **مقدار پیش‌فرض:** `''` (خالی)
    -   **توضیحات:** رنگ آیکون اختاپوس (octocat) را مشخص می‌کند. اگر این فیلد خالی باشد، افزونه به صورت خودکار از رنگ پس‌زمینه سایدبار (`--sidebar-bg`) استفاده می‌کند تا کنتراست کافی ایجاد شود. شما می‌توانید هر مقدار رنگ معتبر CSS (مانند `'#fff'`) را در اینجا وارد کنید.

### مدیریت موقعیت (مهم)

گزینه‌های `position` و `size` از این افزونه حذف شده‌اند. برای کنترل موقعیت، باید از بخش `widgets` در `config.js` استفاده کنید. `id` این ویجت `'github-corner'` است.

**مثال: قرار دادن لینک گیت‌هاب در گوشه بالا-چپ صفحه**
```javascript
// in config.js
window.shirazeh = {
  // ...
  widgets: {
    enabled: true,
    slots: {
      'github-corner': { desktop: 'dock-top-left', mobile: 'hide', priority: 80 },
      // ... سایر ویجت‌ها
    },
  },
  // ...
};
```
> برای اطلاعات کامل در مورد اسلات‌های موجود و گزینه‌ها، به **[مستندات سیستم مدیریت ویجت‌ها](/docs/widgetSystem)** مراجعه کنید.
