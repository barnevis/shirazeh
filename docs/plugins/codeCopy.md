# افزونه رونوشت کد (Code Copy)

این افزونه یک دکمه «رونوشت» به گوشه تمام بلوک‌های کد (`<pre>`) در صفحات شما اضافه می‌کند. این قابلیت به کاربران اجازه می‌دهد تا به راحتی قطعه کدهای موجود در مستندات شما را در کلیپ‌بورد خود رونوشت کرده و استفاده کنند.

## ویژگی‌های اصلی

-   **عملکرد ساده:** با یک کلیک، تمام محتوای بلوک کد رونوشت می‌شود.
-   **بازخورد بصری:** پس از کلیک، متن دکمه به «رونوشت شد!» تغییر می‌کند تا کاربر از موفقیت عملیات مطلع شود.
-   **ظاهر یکپارچه:** استایل دکمه به صورت خودکار با تم (روشن/تاریک) وب‌سایت شما هماهنگ می‌شود.
-   **نمایش هوشمند:** دکمه فقط زمانی نمایش داده می‌شود که کاربر ماوس را روی بلوک کد ببرد تا از شلوغی بی‌مورد صفحه جلوگیری شود.

## فعال‌سازی افزونه

این افزونه به صورت پیش‌فرض در فایل `config/config.js` فعال است. برای فعال یا غیرفعال کردن آن، کافیست مسیر `src/plugins/codeCopy/codeCopy.js` را در آرایه `plugins` نگه دارید یا حذف کنید.

```javascript
// in config/config.js
window.shirazeh = {
  // ...
  plugins: [
    // ... سایر افزونه‌ها
    'src/plugins/codeCopy/codeCopy.js' // فعال‌سازی افزونه رونوشت کد
  ],
  // ...
};
```

## پیکربندی

این افزونه در حال حاضر هیچ گزینه پیکربندی خاصی ندارد و به صورت خودکار روی تمام بلوک‌های کد اعمال می‌شود.
