# افزونه کپی کد (Code Copy)

این افزونه یک دکمه «کپی» به گوشه تمام بلوک‌های کد (`<pre>`) در صفحات شما اضافه می‌کند. این قابلیت به کاربران اجازه می‌دهد تا به راحتی قطعه کدهای موجود در مستندات شما را در کلیپ‌بورد خود کپی کرده و استفاده کنند.

## ویژگی‌های اصلی

-   **عملکرد ساده:** با یک کلیک، تمام محتوای بلوک کد کپی می‌شود.
-   **بازخورد بصری:** پس از کلیک، متن دکمه به «کپی شد!» تغییر می‌کند تا کاربر از موفقیت عملیات مطلع شود.
-   **ظاهر یکپارچه:** استایل دکمه به صورت خودکار با تم (روشن/تاریک) وب‌سایت شما هماهنگ می‌شود.
-   **نمایش هوشمند:** دکمه فقط زمانی نمایش داده می‌شود که کاربر ماوس را روی بلوک کد ببرد تا از شلوغی بی‌مورد صفحه جلوگیری شود.

## فعال‌سازی افزونه

این افزونه به صورت پیش‌فرض در فایل `config/config.js` فعال است. برای فعال یا غیرفعال کردن آن، کافیست مسیر `src/plugins/code-copy/code-copy.js` را در آرایه `plugins` نگه دارید یا حذف کنید.

```javascript
// in config/config.js
window.shirazeh = {
  // ...
  plugins: [
    // ... سایر افزونه‌ها
    'src/plugins/code-copy/code-copy.js' // فعال‌سازی افزونه کپی کد
  ],
  // ...
};
```

## پیکربندی

شما می‌توانید رفتار و متن‌های افزونه را از طریق آبجکت `codeCopy` در فایل `config.js` شخصی‌سازی کنید. اگر این بخش را در فایل تنظیمات خود تعریف نکنید، از مقادیر پیش‌فرض استفاده خواهد شد.

```javascript
// in config/config.js
window.shirazeh = {
  // ...
  codeCopy: {
    buttonText: 'کپی کن',
    successText: 'کپی شد ✔',
    errorText: 'خطا ❌',
    successDuration: 2500,
  },
  // ...
};
```

### گزینه‌های پیکربندی

-   `buttonText`:
    -   **نوع:** `string`
    -   **مقدار پیش‌فرض:** `'رونوشت'`
    -   **توضیحات:** متنی که به صورت پیش‌فرض روی دکمه نمایش داده می‌شود.

-   `successText`:
    -   **نوع:** `string`
    -   **مقدار پیش‌فرض:** `'رونوشت شد'`
    -   **توضیحات:** متنی که پس از موفقیت‌آمیز بودن عملیات کپی، روی دکمه نمایش داده می‌شود.

-   `errorText`:
    -   **نوع:** `string`
    -   **مقدار پیش‌فرض:** `'خطا'`
    -   **توضیحات:** متنی که در صورت بروز خطا در عملیات کپی، روی دکمه نمایش داده می‌شود.

-   `successDuration`:
    -   **نوع:** `number`
    -   **مقدار پیش‌فرض:** `2000`
    -   **توضیحات:** مدت زمانی (به میلی‌ثانیه) که پیام موفقیت یا خطا روی دکمه باقی می‌ماند تا به حالت اولیه بازگردد.
